# Cost Optimization

Observability should not cost more than the infrastructure it monitors. But without deliberate cost control, it easily can.

Metrics are the #1 cost driver. A pragmatic approach — fixing the top 10-20 metrics by series count — typically addresses 60-80% of the problem. This chapter covers how to find and fix cost issues systematically.

## What you'll learn

- Where observability costs actually come from
- The Top-N approach to cost reduction
- Adaptive Metrics for automated optimization
- Label patterns that silently explode your bill
- Log filtering strategies
