# Useful Links

A curated collection of links that every Alloy user should have bookmarked. These are the pages we actually use day-to-day -- not an exhaustive list, but the ones that matter.

Each link is marked with a priority:

- **Essential** -- Bookmark this now. You will use it regularly.
- **Supplementary** -- Useful when you need it. Worth knowing about.

The links are organized into three sections:

1. **[Official Documentation](official-docs.md)** -- The canonical reference pages, installation guides, migration paths, and release notes.
2. **[Community Resources](community.md)** -- Forums, Slack channels, blog posts, video tutorials, example repositories, and conference talks.
3. **[Dashboards and Tools](dashboards-and-tools.md)** -- Pre-built Grafana dashboards, the Alloy Web UI, CLI utilities, editor extensions, and related projects.

> **A note on link rot:** URLs change. If a link below is broken, try searching the Grafana docs site directly at `https://grafana.com/docs/alloy/latest/`. The section structure of the official docs is stable even when individual page URLs move between releases.
