The Alloy Web UI

Alloy ships with a built-in web interface for inspecting running pipelines. It listens on port 12345 by default and is one of Alloy's most underappreciated features.

LinkWhat it coversPriority
HTTP endpoints referenceComplete list of endpoints: / (UI), /metrics (Prometheus scrape), /ready, /live, and more.Essential
Debug AlloyHow to use the UI to inspect component health, view the pipeline graph, and diagnose issues.Essential
CLI run commandThe --server.http.listen-addr flag controls the UI address (default 127.0.0.1:12345). Set to 0.0.0.0:12345 in Docker to expose externally.Essential
Live debugging config blockEnable the live debugging view to see real-time telemetry flowing through each component. Added in v1.3.Essential

Quick access: Once Alloy is running, open http://localhost:12345 in your browser. You will see a visual graph of your pipeline, component health indicators, and links to per-component detail pages.

CLI Tools

The alloy binary includes several built-in subcommands beyond run.

LinkCommandWhat it doesPriority
CLI reference (overview)All subcommandsLanding page for all CLI documentation.Essential
validatealloy validateValidates an Alloy config file or directory without starting the process. Returns zero exit code if valid, non-zero with diagnostics if not. Supports --config.format for validating converted configs (alloy, otelcol, prometheus, promtail, static).Essential
convertalloy convertConverts configs from Prometheus, Promtail, Grafana Agent Static, or OTel Collector format into Alloy config. Use --bypass-errors for best-effort conversion of large files, --report for a diagnostic report.Essential
fmtalloy fmtFormats Alloy config files according to standard conventions. Run this before committing configs.Essential
toolsalloy toolsUtilities for reading the Write-Ahead Log (WAL) and gathering statistical information.Supplementary
otelalloy otelStarts Alloy with the experimental OTel Engine using an OTel Collector YAML config directly.Supplementary
completionalloy completionGenerates shell completion scripts for bash, zsh, fish, and PowerShell.Supplementary
